depth image and point cloud data viewer
----------------------------------------
pc_view.py          class definition of point cloud viewer, include unit test
                    the viewer can be used to show still point cloud or moving point cloud video
dep_csv_viewer.py   GUI that allow user to select depth image file (csv format) to view
pc_csv_viewer.py    GUI that allow user to select point cloud data file (csv format) to view

dep_bravo.csv       test data file for dep_csv_viewer.py
pc_bunny.csv        test data file for pc_csv_viewer.py


usage
-----
please check the unit test code in the end of pc_view.py
